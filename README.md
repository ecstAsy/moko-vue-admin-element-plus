#### 项目介绍

使用 ***Vue3***、 ***Vite***、 ***Typescript*** 搭配 ***Element-Plus*** 框架，搭建项目, 实现 **按需加载**，和自定义 **theme**

#### 克隆项目到本地

> git clone https://github.com/ecstAsy/moko-vue-admin-element-plus.git
#### 依赖管理工具

- pnpm

安装依赖
> pnpm install

启动项目
> pnpm dev

打包项目
> pnpm build

#### 项目组件 ***Components***

-  **GlobalHeader** (顶部导航)
-  **MenuSideBar**  (菜单)
-  **PageWraper**   (页面包裹组件)
-  **MokoIcon**     (Icon)
-  **MokoSvg**      (Svg)
-  **MokoTable**    (列表)
-  **MokoDialog**   (弹窗)
-  **MokoUpload**   (上传)
-  **MokoFileInfo** (文件展示)
-  **MokoTitle**    (标题)
-  **MokoLink**     (超链接)
-  **MokoExport**   (导出列表数据)
